# 氣功打卡小幫手使用者測試說明

這份文件可用來邀請學員進行小規模測試，協助你在正式推廣前收集使用回饋。

---

## 一、測試目的

這次測試的目標是確認：

- 每日打卡流程是否順暢
- 功法勾選 / 心得紀錄是否好用
- 成就、排行榜、個人統計是否清楚
- 提醒訊息是否能有效鼓勵持續練功

---

## 二、目前提供兩個版本

### 1. LINE 版
適合目前已在 LINE 社群中的學員使用。

### 2. Telegram 版
適合願意嘗試較完整、較結構化打卡流程的學員使用。

---

## 三、希望測試者完成的事情

請測試者在 3 到 7 天內盡量完成以下項目：

1. 每天至少打卡一次
2. 查看自己的統計 / 成就
3. 試著看排行榜
4. 若收到提醒訊息，試著從提醒入口回去打卡
5. 若是 Telegram 版，試著填寫：
   - 今日功法
   - 心得
   - 身體感受

---

## 四、LINE 版使用方式

### 加入方式
- 加入 LINE Official Account
- 開啟 1 對 1 聊天室

### 可測試功能
- `✅ Check-In`
- `📊 My Stats`
- `🏆 Leaderboard`
- `🏆 Weekly Leaderboard`
- `🏆 Monthly Leaderboard`
- `🏆 Quarterly Leaderboard`

### 希望回報的問題
- 有沒有按鈕失效
- deep link 是否正常
- 打卡回覆是否清楚

---

## 五、Telegram 版使用方式

### 加入方式
1. 搜尋 Telegram bot
2. 輸入 `/start`
3. 會看到兩個主要入口：
   - `✅ 開始打卡`
   - `🏮 開啟成就頁`

### 可測試功能
- `/checkin`
- `/mystats`
- `/badges`
- `/achievements`
- `/leaderboard`
- `/weekly`
- `/monthly`
- `/quarterly`
- `/yearly`
- `/method30`
- `/method90`

### Telegram 打卡方式
1. 開啟打卡 Web App
2. 勾選今天練習的功法（可複選）
3. 視需要填寫：
   - 練功心得
   - 身體感受
4. 送出打卡

### 特別要測的點
- 同一天重開後是否能修改原內容
- 打卡後是否顯示連續天數 / 成就解鎖
- 成就頁是否好讀、好看

---

## 六、請學員回報哪些內容

若有問題，請盡量回報以下資訊：

1. 使用的是 LINE 還是 Telegram
2. 問題發生的時間
3. 做了什麼操作
4. 畫面上看到什麼訊息
5. 若方便，附上截圖

例如：

> 我用 Telegram，在今天晚上 8:15 打卡時，送出後顯示失敗，畫面寫著 `載入失敗`，附上截圖。

---

## 七、我們最想知道的回饋

請測試者可以特別分享：

1. 打卡流程是否直覺
2. 功法勾選是否清楚
3. 成就 / 統計是否有激勵感
4. 每日提醒是否有幫助
5. 比較喜歡 LINE 還是 Telegram 版本

---

## 九、建議測試流程

### 第一天
- 先學會加入 bot
- 完成第一次打卡

### 第二天
- 再次打卡
- 看看 streak 是否累加

### 第三天之後
- 看 stats / badges / achievements / leaderboard
- 試著觀察提醒訊息是否真的有幫助

---

## 十、測試結束後建議整理的回饋面向

管理者可把回饋分成：

1. **Bug 類**
- 不能打卡
- webhook / reminder 失敗

2. **體驗類**
- 操作不直覺
- 文字太多 / 太少
- 成就頁不夠清楚

3. **需求類**
- 想要更多功法
- 想加練功時間
- 想看更多歷史紀錄

這樣後續比較容易安排下一版優化。
